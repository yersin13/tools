# PRTNUM Formatter

## Run
```bash
npm install
npm run dev
```

## What it does
- Paste values from Excel, TXT, or a copied column
- Keeps values exactly as pasted (letters, hyphens, numbers)
- Shows item count in input and output
- Formats output in 3 modes:
  - SQL quoted comma
  - Plain comma
  - One per line + comma
